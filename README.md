# terraform-aws-msk

## Module create AWS MSK cluster with sasl/scram auth mechanism

```hcl
module "this" {
  source = "../../"

  name  = "msk"
  kms_key_owner_username = "dasmeta.julia"
  client_authentication = {
    "iam" : false,
    "sasl" : {
      "scram" : true
    },
    "tls" : null,
    "unauthenticated" : false
  }

  kafka_auth = {
    username = "username",
    password = "password"
  }

  vpc_id         = "vpc-000000000000000"
  ingress_access = ["10.0.0.0/16"]
}
```
